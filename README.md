# JobMatch-IA

Plataforma inteligente de matching entre candidatos y ofertas de empleo, potenciada por IA.

## Estructura del proyecto

```
jobmatch-ia/
├── frontend/    # React
├── backend/     # FastAPI (Python)
└── .env.example # Variables de entorno
```

## Configuración

1. Copia `.env.example` a `.env` y rellena las variables.
2. Consulta cada subcarpeta para instrucciones de instalación.
